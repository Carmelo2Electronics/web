Project 30

En este proyecto vamos a mandar correos electronicos desde una interfas grafica de Java usando la libreria javax.mail.

El programa corre en Windows o Raspberry Pi, Cuando se ejecuta en Rasppberry se pueden recolectan valores de temperatura, usando la terjeta de Adafruit Max31855 y un termopar, en nuestro caso usamos: Thermocouple Type-K Glass Braid Insulated Stainless Steel Tip. El circuito de conexiones se muestra abajo.

Hay algunos puntos a tener en cuenta para ejecutar este programa:

  • Nada de eso va a trabajar si la direccion imail del correo que se encarga de mandar los mensajes, no fue habilitada para permitir aplicaciones "menos seguras", usa una cuenta de gmail para hacer esto. Hay videos en Youtube que te dicen como hacerlo.

  • Las cuentas de email que se encargan de recibir los correos, no necesitan de lo anterior.

  • El programa queda sujeto a que los servidores de correo lo atiendan, esto, no siempre es asi. El programa te avisara cuando el envio de un correo a fallado o el caso contrario, cuando se ha enviado. El programa en si no se bloquea, pero, si se queda esperando ser atendido por el servidor de correos.

  • Cuando corra el programa en Raspberry Pi, podras ver todas las funiones del programa en ejecucion, pero, debes de instalar la libreria Pi4J en la Raspberry (ve en YouTube como hacer esto), esta se encarga de manejar la tarjeta de Adafruit.

  • Si tienes activadas las notificaciones de la cuenta o cuentas que reciben los correos te puede llegar una alerta de notificacion, por lo cual, puedes usar el programa como un enviador de alarmas, de hecho, esto se puede configurar en el programa.

  • Hay muchas cosas en las cuales se puede emplear el codigo se este programa, nosotros lo empleamos para monitoriar la temperatura de un refrijarador de laboratorio, mandar una alerta cuanda la temperatura sale de valores termicos establecidos como minimo y maximo, tomar una lectura de temperatura cada determinado tiempo y guardar en un archivo de texto plano, la hora en la que se toma la muestra y la temperatura.

  • Se cuidadoso con los valores introducidos al programa, evita en lo posible, estar mandando correos en un mismo momento

Los códigos de programación están en el repositorio.

Esperamos que te sea muy util como referencia para algun otro proyecto o para su empleo tal y como esta hecho.
